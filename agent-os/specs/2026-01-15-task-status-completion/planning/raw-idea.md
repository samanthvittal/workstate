# Raw Idea: Task Status & Completion

**Feature Name:** Task Status & Completion

**Description:** Enhanced status management with completion toggle, status filtering, completion timestamps, bulk actions, and archive functionality for completed tasks

**Key Capabilities:**
- Task completion toggle (interactive checkboxes on cards)
- Status filtering (active/completed views)
- Completion timestamps (track when tasks were completed)
- "Mark all complete" bulk action for task lists
- Archive completed tasks

**Context:**
- Tasks already have a `status` field with 'active'/'completed' values
- Current checkboxes are displayed but disabled
- This is a P0 high-priority feature to complete the core task lifecycle
- Part of the Workstate task management system
